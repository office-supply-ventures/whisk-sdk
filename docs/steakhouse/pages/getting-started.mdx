# Getting Started

Type-safe queries for Steakhouse-curated DeFi vaults.

:::info
You'll need a Steakhouse API key. [Reach out to Steakhouse](mailto:chefs@steakhouse.financial) to get one.
:::

## Installation

:::code-group

```bash [pnpm]
pnpm add @whisk/steakhouse
```

```bash [npm]
npm install @whisk/steakhouse
```

```bash [yarn]
yarn add @whisk/steakhouse
```

:::

## Quick Start

### 1. Set up your Client

```ts twoslash [client.ts]
import { SteakhouseClient } from "@whisk/steakhouse"

const client = new SteakhouseClient({ // [!code focus]
  apiKey: process.env.STEAKHOUSE_API_KEY!, // [!code focus]
}) // [!code focus]
```

### 2. Fetch Stats & TVL

```ts twoslash [index.ts]
import { SteakhouseClient, getStats } from "@whisk/steakhouse"

const client = new SteakhouseClient({
  apiKey: process.env.STEAKHOUSE_API_KEY!,
})

const stats = await getStats(client) // [!code focus]
stats.uniqueDepositors    // Unique depositor count
stats.tvl.current.totalUsd // Current total TVL
```

### 3. Include Historical TVL

```ts twoslash [index.ts]
import { SteakhouseClient, getStats } from "@whisk/steakhouse"

const client = new SteakhouseClient({
  apiKey: process.env.STEAKHOUSE_API_KEY!,
})

const stats = await getStats(client, { includeHistorical: true }) // [!code focus]
stats.tvl.historical // Daily snapshots for the last 365 days
```
